ZeroKB - The no framework CSS framework

ZeroKB is the CSS framework you want when you don't want a CSS framework. The advent of CSS frameworks such as Bootstrap have been a great boon for design-starved startups, but these frameworks often have some unfortunate side-effects:

  • Webpages and apps are often bloated with framework CSS, often 90% of which isn't even used by the page design.
  • Websites built with these frameworks often all look the same, it's fairly easy to spot a bootstrap startup.
  • Developers don't learn anything about CSS, and when something goes wrong they lack the skills to fix the problems themselves.

ZeroKB is an attempt to replace large CSS frameworks with a simple set of guidelines that will:

  • Allow you to design pages the way you want.
  • Work across all modern browsers.
  • Supports LESS, SASS and any future CSS pre-compilers.
  • Make you a more rounded developer.

Creating containers

Containers are the heart of any layout, a container is simply a block level element used to contain other elements (divs are a traditional choice, however this applies to header, footer and other html5 tags).

Some general rules for containers:

  • position: relative; relative positioning is critical for containers, it means that any sub-elements are positioned relative to their container. This makes containers somewhat modular and maintainable.

  • padding: 10px; Be a pro, give your containers padding.

Centering containers

Centering a container is done by setting left and right margins and giving the container a fixed width.

  • margin-left: auto;
  • margin-right: auto;
  • width: 600px;

Centering text

  • To center horizontally, set text-align: center;
  • To center vertically, set height and line-height to the same value.

Flexible middle columns.

This example shows a simple method for handling the situation where you need to fill the space between a left and right floated element with a flexible column.

  • Parent container MUST have an absolute numeric height, ie: height: 60px;
  • Set column height to 100%: height: 100%;
  • Left and right columns are floated left and right, ie: float: left;
  • Middle column magic: overflow: hidden;
